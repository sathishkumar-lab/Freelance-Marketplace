# Freelance Marketplace Website

## Project Description
This project is a freelance marketplace web application developed as part of the BCA Final Year Project.

## Technologies Used
- HTML
- CSS
- PHP
- MySQL
- XAMPP

## Features
- User Registration & Login
- Post Projects
- Bid System
- Freelancer Dashboard
- Profile Management

## How to Run
1. Install XAMPP
2. Import the database into phpMyAdmin
3. Place the project in htdocs
4. Run localhost/projectname
